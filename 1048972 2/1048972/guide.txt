To find the gadgets:

inside of gdb:
breakpoint at display_file
r file.3
info proc mapping

NOW find the gadgets

enter the gadgets' addresses in the program

still inside of gdb:
info files
find the data segment and enter its address in the program

after you do this, play with the input to make the first gadget appear in the return address of the stack
